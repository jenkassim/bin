## Special characters
- `^` : the beginning
- `$` : the end
- `.` : a single character
- `.*` : zero to many character
- `[]` : character classes
- `[^]` : negation
- `{m}` : repeat m times
- `{m, n}` : repeat m through n times
- `+` : repeat one or more times
- `?` : repeat zero or one times
- `()` : a reference group
- `(?P<name>)` : named reference

### Shorthands
- `\d` : digits [0-9]
- `\s` : whitespace [ \t \n \r \f \b]
- `\w` : alphanumeric [a-zA-Z0-9_]
